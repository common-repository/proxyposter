== Description ==
This plugin will allow administrators to post proxy lists from selected sources. Be kind, share a proxy.

== Installation ==
This section describes how to install the plugin and get it working.

1. Upload the "proxy-poster" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select "Proxy Poster" under the "Posts" menu
